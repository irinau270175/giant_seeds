# Семена гигантов — одностраничный каталог

Одностраничный сайт-каталог для продажи семян гигантских овощей (частный садовод). Вся разметка, стили и скрипты в одном файле **`index.html`** — без сборки, без фреймворков и без npm-зависимостей для работы страницы.

## Как открыть

Дважды щёлкните по `index.html` или откройте его в браузере через «Открыть файл». Для корректной загрузки внешних ресурсов (Three.js с CDN, картинки Unsplash) нужен доступ в интернет.

## Структура проекта

| Файл / папка | Назначение |
|--------------|------------|
| `index.html` | Вся страница: контент, `<style>`, скрипты |
| `IMG_5036_1.JPG` | Локальное фото для hero (можно подключить вместо текущего `src` у `<img class="hero__photo">`) |
| `SKILL.md` | Заметки по дизайну для ИИ/редактора (не обязателен для сайта) |

## Настройки заказа в мессенджере

В начале `<head>` задан объект **`window.SITE_ORDER_CONFIG`**:

- `orderChannel` — `telegram` | `max` | `vk`
- `telegramUsername` — логин Telegram **без** `@`
- `maxOrderBase`, `vkUserId` / `vkGroupId` — под MAX и ВКонтакте (см. комментарии в файле)

Текст заказа формируется из шаблона `orderMessageTemplate` и названия товара с карточки. Если выбрана дата в календаре, к сообщению добавляется строка про удобную дату для связи (через **`window.SITE_CONTACT_DATE`**).

## Каталог и фильтры

Карточки товаров — блоки `<article class="catalog-card" data-category="...">`.  
Фильтры — кнопки с `data-filter`. Новый сорт: скопируйте карточку и при необходимости добавьте кнопку фильтра с новым кодом категории; логику JS менять не нужно.

## Технические особенности

- **Three.js** подключается с CDN (фоновый эффект ShapeBlur в hero при отсутствии `prefers-reduced-motion: reduce`).
- **Календарь** и **блок отзывов** реализованы на чистом HTML/CSS/JS (аналоги React-компонентов без React).
- Тёмная оливковая тема и оранжевые акценты задаются CSS-переменными в `:root`.

## Публикация в интернете

Загрузите `index.html` и при необходимости изображения на любой статический хостинг.  
Проверьте пути к локальным картинкам (hero): для сайта в сети лучше использовать **относительные** пути, например `IMG_5036_1.JPG`, а не `file:///...`.

## Если понадобится React / shadcn / Tailwind

Текущий репозиторий этого **не требует**. Для отдельного приложения на Next.js с shadcn обычно создают проект через CLI, папку `components/ui` и уже туда переносят компоненты из документации shadcn.
